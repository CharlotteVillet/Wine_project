Wine-o-meter API Documentation

Welcome to Wine-o-meter, the best place to test your wine

Endpoints

Prediction, method post

This endpoint allows you to make a prediction of your wine's quality with a score from 0 to 10 based on physicochemical samples.

How to request this endpoint?

The expected input is a JSON with an input key followed by an array of inputs. Each inputs is an array of samples in the following order:

  1. fixed acidity
  2. volatile acidity
  3. citric acid
  4. residual sugar
  5. chlorides
  6. free sulfur dioxide
  7. total sulfur dioxide
  8. density
  9. pH
  10. sulphates
  11. alcohol

JSON input example: 

            
        {
          "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
        }

Expected output: 

            
        {
          "prediction": [6, 6]
        }
